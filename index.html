About BeatBox Studio

BeatBox Studio is a browser-based music production app that allows users to create beats using drum pads, a sequencer, and a piano roll. It is designed for beginners and aspiring producers to experiment with rhythm and sound directly in the browser.

Purpose

This project was created as part of a school assignment to explore web development and interactive design.

Features

  • Drum pads (kick, snare, hi-hat, clap)
  • Sequencer / timeline
  • Piano roll
  • Mixer controls
  • Beat saving and sharing
